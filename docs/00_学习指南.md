# XF-AI-Agent 学习指南

## 0. 学习前的准备

### 0.1 前置知识要求

在开始学习这个项目之前，建议具备以下基础知识：

**编程语言**:
- **Python 3.12+**: 项目后端主要使用 Python
- **JavaScript/TypeScript**: 项目前端使用 React + TypeScript
- **SQL**: 了解基本的 SQL 查询

**技术框架**:
- **FastAPI**: Python Web 框架
- **LangGraph**: AI Agent 编排框架
- **React**: 前端框架
- **PostgreSQL**: 关系型数据库
- **Redis**: 缓存和会话存储

**AI/LLM 相关**:
- **大语言模型**: 了解 LLM 的基本概念
- **Agent**: 了解 AI Agent 的基本概念
- **Prompt Engineering**: 了解提示词工程
- **RAG**: 了解检索增强生成

**系统设计**:
- **微服务架构**: 了解微服务的基本概念
- **异步编程**: 了解异步编程的基本概念
- **事件驱动**: 了解事件驱动架构的基本概念

### 0.2 开发环境准备

**后端环境**:
```bash
# 克隆项目
git clone <项目地址>
cd xf-ai-agent

# 创建虚拟环境
python3.12 -m venv .venv
source .venv/bin/activate

# 安装依赖（使用 uv）
pip install uv
uv sync

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入必要的配置

# 启动后端服务
uv run uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

**前端环境**:
```bash
# 克隆前端项目
cd ../xf-ai-agent-app

# 安装依赖
npm install

# 启动前端开发服务
npm run dev
```

**数据库环境**:
```bash
# 启动 PostgreSQL 服务
# 创建数据库
createdb xf_ai_agent

# 执行数据库初始化
psql -U postgres -d xf_ai_agent -f database_init.sql
```

**Redis 环境**:
```bash
# 启动 Redis 服务
redis-server
```
