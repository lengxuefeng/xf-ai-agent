# XF-AI-Agent 项目说明文档

XF-AI-Agent 是一套基于 **LangChain + LangGraph + FastAPI** 的高级多智能体 (Multi-Agent) AI 对话系统。它不仅具备灵活的任务分发能力，还内置了 RAG（检索增强生成）、人工审批（Human-in-the-loop）、以及高度定制化的 SSE 实时流式输出机制。

---

## 🌟 核心功能特性

1. **多智能体协同 (Multi-Agent Routing)**
   - 通过 `Supervisor` 扮演“总路由大脑”，自动通过 关键词匹配 / LLM语义判断 / 规则兜底 三级机制，将用户的原生意图分发给最合适的领域专家。
   - 目前内置的专家智能体包括：`code_agent` (代码与技术支持)、`sql_agent` (数据库查询与分析)、`medical_agent` (医疗文案与咨询)、`yunyou_agent` (云游业务系统交互)、`search_agent` (联网搜索搜索)。

2. **全栈真·实时流式响应 (True Real-time SSE)**
   - 极致的性能优化：利用 **后台多线程 + 线程安全队列 + Python logging 拦截** 架构，将后端的工具调用、模型初始化、底层网络报错甚至 API 交互日志，以毫秒级延迟呈现至前端的“深度思考过程”面板中。
   - 所有执行异常（如第三方服务 502/Timeout）都将被系统自动捕获，并优雅地格式化为前端的流式报错卡片，告别白屏和未知错误。

3. **人机协同中断审批 (Human-in-the-Loop)**
   - 对于敏感操作（如向真实数据库执行 SQL 查询、进行关键系统改动），Agent 会主动挂起并在前端弹窗，等待用户确认（Approve/Reject）后，携带审批指令从断点继续执行。

4. **灵活的模型与记忆管理**
   - 通过 `UnifiedModelLoader` 统一接入海量大模型生态：天然支持 OpenAI 系列、Gemini 系列、通义千问模型、甚至是本地部署的 Ollama 等等。
   - 与 PostgreSQL 和 MongoDB 深度融合：使用 PgSQL+LangGraph Checkpointer 保存智能体会话图谱的上下文断点；使用 MongoDB 持久化存取用户的历史聊天记录。

5. **企业级鉴权与安全性**
   - 基于 JWT Bearer 的无状态 Token 验证体系。
   - 完善的 401 / 失效 Token 处理机制，前后端闭环。

---

## 🏗️ 整体系统架构

系统采用清晰的前后端分离架构，通过 REST API 与 Server-Sent Events (SSE) 进行核心通信。

- **后端 (`xf-ai-agent`)：** 采用 Python 编写，基于 FastAPI 框架提供极速的异步 I/O，业务工作流引擎全面拥抱 LangGraph。包含精密的路由层、工具注册表和流式编排中心。
- **前端 (`xf-ai-agent-app`)：** 采用 React 18 + Vite 构建。自定义封装了一套高鲁棒性的 SSE 接收解析库，不仅渲染最终的 Markdown 内容，而且能在侧边栏展示多模态的“思考过程”及错误拦截面板。

## 🛠️ 本地开发与快速启动

### 1. 环境准备
- **数据库**: 需要准备 PostgreSQL (用于 LangGraph 状态机) / MongoDB (可选，用于完整记忆) / PGVector (可选，用于 RAG 知识库)。
- **Python**: 推荐 Python 3.10+
- **Node.js**: 推荐 Node.js 18+

### 2. 后端部署
```bash
cd xf-ai-agent

# 安装依赖
pip install -r requirements.txt

# 复制一份环境变量并填入大模型API-Key与数据库链接
cp .env.example .env

# 启动 FastAPI 服务
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```
后端服务默认启动在 `http://localhost:8000`，自动生成 Swagger 接口文档可用位于 `/docs`。

### 3. 前端部署
```bash
cd xf-ai-agent-app

# 安装依赖包
npm install

# 配置指向本地后端的环境变量
echo "VITE_API_URL=http://localhost:8000/api/v1" > .env.local

# 启动 Vite 本地开发服务器
npm run dev
```
